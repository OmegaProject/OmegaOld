Folder contents:
================

\ddl:					            SPT DLL and wrapper 
 
 \omega-spt-wrapper-dll.zip 		This VS 2010 project creates a DLL wrapper that can be used to call the SPT DLL
 
 \ParticleTrackerDll_Statistics.zip	Zip file containing the SPT DLL
 
 \INFO.txt							A file containing information about the last DLL updates.		



\manuals:               		The usage manuals

 \ParticleTracker.pdf   		Describes the parameters ('Ps') of the Single Particle Tracker (SPT) algorithm



\test_track:            		A set of trajectories that can be used to test the automatic Trajectories 
                        		Segmentation (TS) algorithm



\trained_patterns:      		The trained patterns for the direct motion and the drift motion. Use these 
                        		patterns when running the automatic Trajectories Segmentation (TS) algorithm
